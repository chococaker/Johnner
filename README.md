## Features

- Magic bitboards
- Minimax w/ alpha-beta pruning
- Barebones UCI implementation
- Piece-Square Tables
- Transposition tables (Zobrist hashing)
- Quiescence Search
- Late-move reduction
- Move ordering

## Building

Requires CMake.

Run `build_johnner.bat`, and grab `johnner_uci.exe`. This is a UCI engine. Have fun!
